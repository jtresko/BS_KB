# Step 1

Join the BigShort Discord server [here](https://discord.com/invite/6w22gAVzMB). Be sure to join with the same Discord username you want to access the members-only channels with, we allow one username per account.

# Step 2

Sign into app.bigshort.com, then click on the down caret in the upper right-hand corner, and select "Members Discord."
